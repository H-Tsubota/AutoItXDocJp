Method Reference (COM)

ToolTip

スクリーンの任意の場所にツールチップを作成します。

ToolTip "text" [, x, y]

 

パラメータ

text ツールチップのテキスト (空の文字列を指定するとツールチップの表示を終了)
x, y ツールチップのx、y座標

 

返し値

なし。

 

注意

x、y座標を省略した場合、チップはマウスカーソルの近くに表示されます。座標によってツールチップがスクリーンからはみ出す場合は表示される様に再配置されます。
ツールチップはクリアされるか、スクリプトが終了するか、クリックされるまで表示され続けます。ラインフィード文字を使用することで複数行のテキストを持つツールチップを作成できます。

 

関連

TrayTip

 

例

Set oAutoIt = WScript.CreateObject("AutoItX3.Control")

' スクリーンの左上にツールチップを作成します。
oAutoIt.ToolTip "This is a tooltip", 0, 0
oAutoIt.Sleep 2000     ' Sleep to give tooltip time to display
