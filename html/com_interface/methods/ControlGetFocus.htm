Method Reference (COM)

ControlGetFocus

指定したウィンドウ内でキーボードフォーカスのあっているコントロールのControlRef#を返します。

ControlGetFocus "title" [, "text"]

 

パラメータ

title 調べるウィンドウのタイトル
text オプション：調べるウィンドウのテキスト

 

返し値

成功: 指定したウィンドウ内でキーボードフォーカスのあっているコントロールのControlRef#を返します。
失敗: ウィンドウが見つからなかった場合、空の文字列を返し、 oAutoIt.errorを1に設定します。

 

注意

なし。

 

関連

ControlFocus, ControlCommand, WinActive

 

例

Set oAutoIt = WScript.CreateObject("AutoItX3.Control")
a = oAutoIt.ControlGetFocus("Untitled - Notepad")
