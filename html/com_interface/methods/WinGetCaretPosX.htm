Method Reference (COM)

WinGetCaretPosX

最前面ウィンドウ内のキャレットの位置を返します。

WinGetCaretPosX

 

パラメータ

なし。

 

返し値

成功: キャレットのX座標を返します。
失敗: oAutoIt.errorを1に設定

 

注意

WinGetCaretPosはCaretCoordModeで絶対値が設定されてる場合、マルチプルドキュメントインターフェイス(MDI)アプリケーションから正確な値を返しません。この問題の回避方法については例を参照してください。注意：アプリケーションによってはキャレット位置ではなくスタティックコントロール位置を返します！

 

関連

CaretCoordMode (Option)

 

例

Set oAutoIt = WScript.CreateObject("AutoItX3.Control")
x = oAutoIt.WinGetCaretPosX()
y = oAutoIt.WinGetCaretPosY()
