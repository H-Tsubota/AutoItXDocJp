Method Reference (COM)

WinActivate

ウィンドウをアクティブにします（ウィンドウをフォーカスします）。

WinActivate "title" [, "text"]

 

パラメータ

title アクティブにするウィンドウのタイトル
text オプション： アクティブにするウィンドウのテキスト

 

返し値

なし。

 

注意

WinActiveを使えばＷｉｎＡｃｔｉｖａｔｅが成功したかどうか確認することができます。もし複数のウィンドウが処理の対象に該当する場合、もっともアクティブなウィンドウがアクティブ化されます。WinActivateは最小化されたウィンドウにも有効です。しかし”常に最前面”に設定されたウィンドウより前にだすことはできません。

 

関連

WinClose, WinSetState, WinTitleMatchMode (Option)

 

例

Set oAutoIt = WScript.CreateObject("AutoItX3.Control")
oAutoIt.WinActivate "Untitled - Notepad", ""
