Method Reference (COM)

CDTray

CDトレイの開閉をします。

CDTray "drive", "status"

 

パラメータ

drive 制御するCDトレイのドライブ名。D:、E:などのように記述します。
status CDトレイを開けるのか閉じるのかの指定。"open"または"closed"。

 

返し値

成功: 1を返します。
失敗: ドライブがCD用ソフトウェアによってロックされている場合、指定したドライブがCDドライブでない場合は0を返します。

 

注意

CDTrayはDAEMONツールのような仮想CDドライブを想定して動作します。
CDTrayはローカルでなかったり、マップされていないCDドライブに対しては動作しません。CDTrayはドライブの存在するコンピュータから実行されなければなりません。
ラップトップパソコンにみられる手動でしか閉じられないCDトレイに対してもCDTray("X:", "close") は0を返します。

 

関連

なし。

 

例

' ドライブEのCDトレイを開く
Set oAutoIt = WScript.CreateObject("AutoItX3.Control")
oAutoIt.CDTray "E:", "open"
