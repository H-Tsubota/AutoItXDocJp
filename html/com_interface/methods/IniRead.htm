Method Reference (COM)

IniRead

標準形式の.iniファイルから値を読み取ります。

IniRead "filename", "section", "key", "default"

 

パラメータ

filename .iniファイルのファイル名
section .iniファイル内のセクション名
key .iniファイル内のキー名
default 要求したキーが見つからなかった場合、返されるデフォルト値

 

返し値

成功: 要求したキーの値が返されます。
失敗: 要求したキーが見つからなかった場合、default文字列が返されます。

 

注意

標準的なINIファイルは次のようになっています。
[Section名]
Key=Value

 

関連

IniDelete, IniWrite, FileReadLine

 

例

Set oAutoIt = WScript.CreateObject("AutoItX3.Control")
var = IniRead "C:\Temp\myfile.ini", "section2", "key", "NotFound"
WScript.Echo "Result: " & var
