Method Reference (COM)

IsAdmin

現在のユーザーが完全な管理者権限を持っているかどうか調べます。

IsAdmin

 

パラメータ

なし。

 

返し値

成功: 現在のユーザーが管理者権限を持っている場合1を返します。
失敗: 管理者権限を持っていない場合0を返します。

 

注意

この関数はWindow 95/98/Meでは常に1を返します。

 

関連

RunAsSet

 

例

Set oAutoIt = WScript.CreateObject("AutoItX3.Control")

If oAutoIt.IsAdmin Then WScript.Echo "Admin rights detected"
