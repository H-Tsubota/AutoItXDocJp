Method Reference (COM)

MouseWheel

マウスホイールを上下させます。NT/2000/XP のみ対応。

MouseWheel "direction" [, clicks]

 

パラメータ

direction "up" または "down"
clicks オプション：ホイールを動かす回数。デフォルトは1。

 

返し値

なし。

 

注意

directionパラメータを認識できなかった場合、oAutoIt.errorを1に設定します。

この関数はNT, 2000, XP以降のOSでのみ動作します。

 

関連

MouseClick, MouseClickDrag, MouseGetPosX, MouseGetPosY, MouseMove, MouseCoordMode (Option)

 

例

Set oAutoIt = WScript.CreateObject("AutoItX3.Control")

' ホイールを上に10回動かします。
oAutoIt.MouseWheel "up", 10
