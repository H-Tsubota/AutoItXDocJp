Method Reference (COM)

DriveMapAdd

ネットワークドライブをマップします。

DriveMapAdd "device", "remote share" [, flags [, "user" [, "password"]]]

 

パラメータ

device マップするデバイス。例："O:"、"LPT1:"。このパラメータに空文字列を設定するとコネクションは作られますが特定のドライブへのマップはおこなわれません。"*"を指定すると使用していないドライブ文字が自動で選択されます。
remote share 接続するリモートの共有フォルダ。"\\server\share"形式で指定。
flags オプション：
次のものを組み合わせられる。
0 = デフォルト
1 = 永続的なマッピング
8 = 必要な場合、認証ダイアログを表示
user オプション： 接続時に使用するユーザー名。"username"形式か、"domain\username"形式のいずれか。
password オプション： 接続時に使用するパスワード。

 

返し値

成功: 1を返します。(注意を参照)
失敗: 新しいマッピングを作成できなかった場合、0を返しoAutoIt.errorを設定します(下記を参照)。(注意を参照)

 

注意

関数が失敗した場合(0を返し)、oAutoIt.errorに情報が設定されます。
 1 = 未定義/その他のエラー。
 2 = リモート共有へのアクセスが拒否されました。
 3 = デバイスは既に割り当てられています。
 4 = 無効なデバイス名
 5 = 無効なリモート共有
 6 = 無効なパスワード

注意：deviceパラメータに"*"を使用すると選択されたドライブ文字(例： "U:")が1、0の代わりに返されます。"*"使用時にエラーが起きた場合は空文字列""が返されます。

 

関連

DriveMapDel, DriveMapGet

 

例

Set oAutoIt = WScript.CreateObject("AutoItX3.Control")

' ドライブXを\\myserver\stuffに現在のユーザーでマップ
oAutoIt.DriveMapAdd "X:", "\\myserver\stuff"

' ドライブXを\\myserver2\stuff2にユーザー"jon"、"domainx"形式、パスワード"tickle"でマップ
oAutoIt.Drivemap "X:", "\\myserver2\stuff2", 0, "domainx\jon", "tickle"
