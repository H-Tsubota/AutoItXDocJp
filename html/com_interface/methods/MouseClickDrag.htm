Method Reference (COM)

MouseClickDrag

マウスのクリックとドラッグ操作をおこないます。

MouseClickDrag "button", x1, y1, x2, y2 [, speed]

 

パラメータ

button クリックするボタン： "left"、"right"、"middle"、"main"、"menu"、"primary"、"secondary"を指定できます。
x1, y1 ドラッグ開始位置のx/y座標
x2, y2 ドラッグ終了位置のx/y座標
speed オプション：1（最速）から100（最低速）で指定されるマウスの移動スピード。0を指定するとただちに移動します。デフォルトのスピードは10。

 

返し値

なし。

 

注意

buttonパラメータに空の文字列を指定すると左ボタンがクリックされます。

buttonパラメータがリストにない場合、oAutoIt.errorを1に設定します。

コントロールパネルでマウスの右と左のボタン機能を交換した場合ボタンの働きが変わります。"Left"、"right"は交換されているかどうかに関らず常にそのボタンをクリックします。"primary" 、"main"ボタンは交換されているかどうかに関らずメインクリックをおこないます。 "secondary"、"menu"ボタンは交換されているかどうかに関らずコンテキストメニューの表示をおこないます。詳しくはMouseClick の表を参照

 

関連

MouseClick, MouseGetPosX, MouseGetPosY, MouseMove, MouseCoordMode (Option)

 

例

Set oAutoIt = WScript.CreateObject("AutoItX3.Control")

; 左ボタンをクリックして（0,200）から（600, 700）へドラッグします。
oAutoIt.MouseClickDrag "left", 0, 200, 600, 700
