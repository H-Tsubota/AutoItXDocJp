Method Reference (COM)

DriveMapDel

ネットワークドライブを切断します。

DriveMapDel "device"

 

パラメータ

drive 切断するデバイス。例："O:"、"LPT1:"。

 

返し値

成功: 1を返します。
失敗: 切断に失敗した場合0を返します。

 

注意

マップされているドライブ文字がない場合、接続名(例：\\server\share)を切断に使用できます。

 

関連

DriveMapAdd, DriveMapGet

 

例

Set oAutoIt = WScript.CreateObject("AutoItX3.Control")

' ドライブXを\\myserver\stuffに現在のユーザーでマップ
oAutoIt.DriveMapAdd "X:", "\\myserver\stuff"

' 切断
oAutoIt.DriveMapDel "X:"
