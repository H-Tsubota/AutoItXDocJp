Method Reference (COM)

ClipGet

クリップボードからテキストを取得します。

ClipGet

 

パラメータ

なし。

 

返し値

成功: クリップボード上のテキストを文字列として返します。
失敗: クリップボードが空の場合、テキストでない内容の場合oAutoIt.errorを1に設定します。

 

注意

なし。

 

関連

ClipPut

 

例

Set oAutoIt = WScript.CreateObject("AutoItX3.Control")

text = oAutoIt.ClipGet()
WScript.Echo "Clipboard contains:" & text
