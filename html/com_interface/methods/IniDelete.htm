Method Reference (COM)

IniDelete

標準形式の.iniファイルから値を削除します。

IniDelete "filename", "section" [, "key"]

 

パラメータ

filename .iniファイルのファイル名
section .iniファイル内のセクション名
key オプション：.iniファイル内の削除するキー名。キー名を指定しない場合、セクション全体が削除されます。

 

返し値

成功: 1を返します。
失敗: INIファイルが存在しない場合、読み取り専用の場合は0を返します。

 

注意

標準的なINIファイルは次のようになっています。
[Section名]
Key=Value
filenameが見つからない場合、0バイトのファイルが作成されます。

 

関連

IniRead, IniWrite

 

例

Set oAutoIt = WScript.CreateObject("AutoItX3.Control")
oAutoIt.IniDelete "C:\Temp\myfile.ini", "section2", "key"
