Method Reference (COM)

MouseGetPosY

マウスカーソルの現在のY座標を取得します。

MouseGetPosY

 

パラメータ

なし。

 

返し値

マウスカーソルの現在のY座標を返します。

 

注意

相対座標/絶対座標の設定についてはMouseCoordModeを参照してください。相対座標モードの場合、座標が負になることもあります。

 

関連

MouseClick, MouseClickDrag, MouseGetPosX, MouseMove, MouseCoordMode (Option)

 

例

Set oAutoIt = WScript.CreateObject("AutoItX3.Control")

posx = oAutoIt.MouseGetPosX()
posy  = oAutoIt.MouseGetPosX()

WScript.Echo "Mouse x,y:" & posx & "," & posy
