Method Reference (COM)

DriveMapGet

マップされたドライブの内容を取得します。

DriveMapGet"device"

 

パラメータ

device 調べるデバイス(ドライブ、プリンター)文字。例："O:"、"LPT1:"。

 

返し値

成功: マップの内容を返します。例：\\server\share
失敗: 空文字列 ""を返し、oAutoIt.errorを1に設定します。

 

注意

なし。

 

関連

DriveMapAdd, DriveMapDel

 

例

Set oAutoIt = WScript.CreateObject("AutoItX3.Control")

' ドライブXを\\myserver\stuffに現在のユーザーでマップ
oAutoIt.DriveMapAdd "X:", "\\myserver\stuff"

' マップの内容を取得
WScript.Echo "Drive X: is mapped to " & DriveMapGet("X:")
