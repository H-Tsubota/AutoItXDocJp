Method Reference (COM)

WinExists

指定したウィンドウが存在するかどうかを調べます。

WinExists "title" [, "text"]

 

パラメータ

title 調べるウィンドウのタイトル
text オプション：調べるウィンドウのテキスト

 

返し値

ウィンドウが存在する場合1を返します。それ以外の場合0を返します。

 

注意

ウィンドウが存在する場合、たとえ隠されていても1を返します。

 

関連

WinActive, WinWait, WinWaitActive, WinWaitClose, WinWaitNotActive, WinTitleMatchMode (Option)

 

例

Set oAutoIt = WScript.CreateObject("AutoItX3.Control")

If oAutoIt.WinExists("Untitled -") Then
    WScript.Echo "Window exists"
EndIf
