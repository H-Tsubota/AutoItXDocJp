Method Reference (COM)

WinGetTitle

ウィンドウの完全な名前を取得します。

WinGetTitle "title" [, "text"]

 

パラメータ

title 対象となるウィンドウのタイトル
text オプション：対象となるウィンドウのテキスト

 

返し値

完全なウィンドウ名を格納した文字列を返します。ウィンドウが見つからなかった場合、数値の0を返します。

 

注意

WinGetTitle("") はアクティブなウィンドウの名前を返します。WinGetTitleは最小化されたウィンドウ、隠されたウィンドウ両方ともに有効です。もし複数のウィンドウが処理の対象に該当する場合、もっともアクティブなウィンドウが処理されます。

 

関連

AutoItWinGetTitle, WinGetText, WinSetTitle, WinTitleMatchMode (Option)

 

例

Set oAutoIt = WScript.CreateObject("AutoItX3.Control")

title = oAutoIt.WinGetTitle "Untitled -", ""
WScript.Echo "Full title read was:" & title
