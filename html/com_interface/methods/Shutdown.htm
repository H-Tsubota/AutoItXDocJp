Method Reference (COM)

Shutdown

システムをシャットダウンします。

Shutdown ( code )

 

パラメータ

code シャットダウンコードの組み合わせ。注意を参照。

 

返し値

成功: 1を返します。
失敗: 0を返します。

 

注意

シャットダウンコードには次の値を組み合わせて使用することができます。
0 = ログオフ(Logoff)
1 = シャットダウン(Shutdown)
2 = 再起動(Reboot)
4 = 強制(Force)
8 = パワーダウン(Power down)

使用したい値を足し合わせてください。例えばシャットダウンとパワーダウンをおこないたい場合はコードは9になります(シャットダウン + パワーダウン = 1 + 8 = 9)

スタンバイと休止はhttp://grc.com/wizmo/wizmo.htmと言ったサードパーティ製のソフトウェアを使用することで可能です。
