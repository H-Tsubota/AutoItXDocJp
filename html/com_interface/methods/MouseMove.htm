Method Reference (COM)

MouseMove

マウスポインタを動かします。

MouseMove x, y [, speed]

 

パラメータ

x マウス移動先のスクリーンx座標
y マウス移動先のスクリーンy座標
speed オプション： 1（最速）から100（最低速）で指定されるマウスの移動スピード。0を指定するとただちに移動します。デフォルトのスピードは10。

 

返し値

連続的なMouseMove処理中にはユーザーのマウス操作は妨げられます。
MouseCoordModeが相対座標モードの場合、座標が負になることもあります。

 

注意

なし。

 

関連

MouseClick, MouseClickDrag, MouseGetPosX, MouseGetPosY, MouseCoordMode (Option)

 

例

Set oAutoIt = WScript.CreateObject("AutoItX3.Control")

oAutoIt.MouseMove 10, 100
oAutoIt.MouseMove 700, 700, 0
