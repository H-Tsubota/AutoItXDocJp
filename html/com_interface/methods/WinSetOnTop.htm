Method Reference (COM)

WinSetOnTop

ウィンドウの”Always On Top（常に最前面に表示）” 属性を変更します。

WinSetOnTop "title", "text", flag

 

パラメータ

title 対象となるウィンドウのタイトル
text 対象となるウィンドウのテキスト
flag ウィンドウが”TOPMOST”フラグを設定します。
1=最前面表示、0 = 最前面表示を解除

 

返し値

なし。

 

注意

サードパーティーのプログラムの”Always On Top”指定がAutoItの指定したTOPMOSTフラグと対立した場合AutoItが優先されます。

 

関連

WinSetState

 

例

Set oAutoIt = WScript.CreateObject("AutoItX3.Control")
oAutoIt.WinSetOnTop "Untitled -", "", 1
