Method Reference (COM)

MouseDown

現在のマウス位置でボタンを押す動作をおこないます。

MouseDown "button"

 

パラメータ

button クリックするボタン： "left"、"right"、"middle"、"main"、"menu"、"primary"、"secondary"を指定できます。

 

返し値

なし。

 

注意

MouseClickの注意欄を参照。
ユーザーは全てのMouseDownごとにMouseUpをおこなわなければいけません。

 

関連

MouseUp, MouseClick, MouseClickDrag, MouseGetPosX, MouseGetPosY, MouseMove, MouseCoordMode (Option)

 

例

Set oAutoIt = WScript.CreateObject("AutoItX3.Control")

oAutoIt.MouseDown "left"
oAutoIt.Sleep 100
oAutoIt.MouseUp "left"
