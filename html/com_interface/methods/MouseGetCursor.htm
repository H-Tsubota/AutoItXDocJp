Method Reference (COM)

MouseGetCursor

現在のマウスカーソルのカーソルIDを返します。

MouseGetCursor

 

パラメータ

なし。

 

返し値

カーソルIDを返します。
    0 = UNKNOWN (不明) (ポインティング、手掴みアイコンを含む)
    1 = APPSTARTING (アプリケーション開始)
    2 = ARROW (矢印)
    3 = CROSS (十字)
    4 = HELP (ヘルプ)
    5 = IBEAM (I-字型)
    6 = ICON (アイコン)
    7 = NO (禁止)
    8 = SIZE (サイズ)
    9 = SIZEALL (全方向)
  10 = SIZENESW (北東-南西方向サイズ)
  11 = SIZENS (南北方向サイズ)
  12 = SIZENWSE (北西-南東方向サイズ)
  13 = SIZEWE (東西方向サイズ)
  14 = UPARROW (上矢印)
  15 = WAIT (待機)

 

注意

なし。

 

関連

MouseGetPosX, MouseGetPosY

 

例

Set oAutoIt = WScript.CreateObject("AutoItX3.Control")

oAutoIt.Sleep 2000  'allow time to move mouse before reporting ID
cursor = MouseGetCursor()
WScript.Echo "ID = " & cursor
